### 사용자 맞춤 병역특례공고 제공 서비스
* 병무청에 등록된 공고를 제공합니다.

* 직군별·지역별 병역특례공고를 제공합니다.
* 사용자의 이력서와 자주 클릭한 공고를 분석해 맞춤 공고를 추천해드립니다.
